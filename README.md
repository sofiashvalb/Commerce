# Commerce Web App

**A full-stack web application for buying and selling items online. Users can create listings, browse products, manage a personal watchlist, and filter items by category—all in a clean, responsive interface built with Django and front-end JavaScript.**

---

## Features

- **User Authentication** – Register, log in, and log out securely
- **List & Purchase Items** – Users can create item listings and mark them as sold
- **Watchlist** – Save items to a personal watchlist for quick access
- **Category Filtering** – Browse items by category with real-time filtering
- **Responsive UI** – Built with mobile-friendly HTML, CSS, and JavaScript

---

## Tech Stack

- **Backend:** Django (Python)
- **Database:** SQLite (via Django ORM)
- **Frontend:** HTML, CSS, JavaScript
- **Authentication:** Django's built-in User system

---

## Highlights

- **Dynamic Listings:**  
  New items appear in real-time on the homepage and are filterable by category

- **Watchlist System:**  
  Users can quickly add or remove items from their saved list

- **Custom Views & Templates:**  
  Each item has a detailed listing page with purchase status, description, and seller info

- **Seamless User Flow:**  
  From account creation to checkout, the experience is smooth and intuitive
